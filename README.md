# Le Safir et le Roi

Game created over a week-end, during the GameJam of *"Les Rendez-vous de l'Histoire 2025"*.
You can access all the game created during the event on this link : https://itch.io/jam/game-jam-des-rendez-vous-de-lhistoire-2025.

## Pitch

*Le Safir et le Roi* is a narrative game where you follow a snapshot of *Ousama Ibn Mounqidh*, a syrian ambassador on the XII<sup>th</sup> century.
Though a trial like game, *Ousâma* and you will have to choose arguments to plead a theft case with the rulor of Jerusalem, the *King Foulques*, and his court of jurors.
